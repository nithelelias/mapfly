# ✈️ MapFly

MapFly is a browser-based flight route game where you pilot an aircraft across a world map.
Choose destinations, plan routes between cities, manage fuel, and take off to complete your flights.

The game is built with a lightweight web stack and focuses on interactive map-based gameplay.

## 🕹️ Gameplay

In MapFly you:

* Control an aircraft on a global map
* Select routes between cities
* Analyze distance and rewards
* Manage fuel before takeoff
* Choose the most profitable flight plan

Every route is a strategic decision between **distance, fuel cost, and reward**.

---

## 🗺️ Features

* Interactive world map
* City-to-city flight routes
* Route distance calculation 
* Route selection with visual highlighting
* Simple flight planning interface

---

## 🛠️ Tech Stack

* **Vite** – fast development environment
* **TypeScript** – typed JavaScript
* **Leaflet** – interactive maps
* **OpenStreetMap** – map data
* **geonames** – api for city data.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/mapfly.git
cd mapfly
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
